Projects & Reports

Classifier Rules to Solve the Majority Problem

Simulated all elementary automata, studied its statistical mechanics and applications in solving the majority problem (a density classification task), both theoretically and as a proposed model to explain the less understood phenomenon of stomatal patchiness.

Face-mask Detection using Deep Neural Networks

An efficient face-mask detection tool suitable for cheap computation such as on mobile phones, webcams or CCTV cameras, so as to allow for real-time feedback into disease dynamics models.

Modelling Call Pattern Generation in Bushcrickets

Exploratory project on call pattern generation in crickets and mechanistic models of neurons, particularly the Morris-Lecar 2-neuron model. Project was done as part of an Independent Study Module with Bittu Rajaraman.

Prediction of Erroneous Decision Making

Decoded neural data from Steinmetz, et. al, 2019, to predict erroneous decision making in trained mice. Used a GLM to predict whether the performance of trained mice that performed well in an 2-AUC experimental paradigm was affected by previous erroneous decisions.
